If you split 192.168.1.0/24 into two equal subnets, you borrow 1 bit from the host portion:

/24 → /25

Subnet mask: 255.255.255.128

!Note: You borrow 1 bit from the host portion so the 
subnet mask will be 11111111.11111111.11111111.10000000, 128(10000000).

Resulting sub-networks
1️⃣ Subnet 1
Network: 192.168.1.0/25
Usable IP range: 192.168.1.1 – 192.168.1.126
Broadcast: 192.168.1.127
Usable hosts: 126

2️⃣ Subnet 2
Network: 192.168.1.128/25
Usable IP range: 192.168.1.129 – 192.168.1.254
Broadcast: 192.168.1.255
Usable hosts: 126
---------------------------------------------------------------------------------------------------------------------
Starting point
192.168.1.0/24

You choose:
/26

That means you borrowed 2 bits from the host portion:
/24 + 2 = /26

How many networks does /26 create?

Formula:
2^borrowed_bits = number of subnets
2^2 = 4 subnets ✅

So /26 → 4 networks ✔️

Subnet mask for /26
/26 = 255.255.255.192

Why?
192 = 11000000

Block size (this tells you where networks start)
Block size = 256 − 192 = 64

So each subnet jumps by 64 in the last octet.

The 4 sub-networks

1️⃣ Subnet 1
Network: 192.168.1.0/26
Usable: 192.168.1.1 – 192.168.1.62
Broadcast: 192.168.1.63

2️⃣ Subnet 2
Network: 192.168.1.64/26
Usable: 192.168.1.65 – 192.168.1.126
Broadcast: 192.168.1.127

3️⃣ Subnet 3
Network: 192.168.1.128/26
Usable: 192.168.1.129 – 192.168.1.190
Broadcast: 192.168.1.191

4️⃣ Subnet 4
Network: 192.168.1.192/26
Usable: 192.168.1.193 – 192.168.1.254
Broadcast: 192.168.1.255

4 * 62 = 248 total hosts possible including 4 subnets.

If the ips are 192.168.1.X/26 and
255.255.255.192 is the subnet mask then the after performing and operation
with the ips the result will be
192.168.1.0/26 (Applied on subnet 1)
192.168.1.64/26 (Applied on subnet 2)
192.168.1.18/26 (Applied on subnet 3)
192.168.1.192/26 (Applied on subnet 4)
