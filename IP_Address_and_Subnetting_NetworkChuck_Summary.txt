Detailed Summary — what is an IP Address?

1. What an IP Address Is
An IP address is a unique identifier assigned to every device on a network. It allows devices to send and receive data. Any device that connects to a network—computers, phones, smart devices—must have an IP address to communicate.

2. How Devices Get IP Addresses
Most devices receive their IP address automatically from a DHCP server, which is usually your router. This process happens when a device joins a network, removing the need for manual configuration in most home and office setups.

3. Structure of an IP Address
An IPv4 address consists of four numbers (octets) separated by dots, such as 192.168.1.15. Part of the address represents the network, and the remaining part identifies the individual device (host). Devices on the same local network share the same network portion.

4. Subnet Masks
A subnet mask defines which part of the IP address belongs to the network and which part belongs to the host. A common subnet mask is 255.255.255.0, meaning the first three octets are the network portion and the last octet identifies individual devices.

5. Default Gateway and Routing
When a device wants to communicate with another device on the same network, it sends traffic directly. When it wants to communicate with something outside the network, the traffic is sent to the default gateway (the router), which knows how to reach other networks, including the internet.

6. Usable IP Addresses
In a network using 255.255.255.0, there are 256 total addresses. However:
- The first address is the network address
- The last address is the broadcast address
- One address is usually assigned to the router
This leaves roughly 250 usable addresses for devices.
